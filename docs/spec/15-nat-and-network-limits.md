# 15 NAT 与网络限制

定义 jiayi-fsp 在家庭网络环境下的支持范围与限制。

## 15.1 支持场景

| 场景        | 支持情况   |
| --------- | ------ |
| 单一局域网     | 完全支持 |
| 二级路由（NAT） | 条件支持 |
| 跨公网 NAT   | 不支持  |
| 移动蜂窝网络    | 不支持  |

## 15.2 网络假设

- Server 与 Client 逻辑处于同一局域网
- Client 可主动向 Server 发起 TCP 与 UDP 流量
- 协议不提供公网穿透能力（无 STUN/TURN）

## 15.3 兼容建议

- Client 先发 UDP 流量建立 NAT 映射
- mDNS 发现可能失效，需提供手动配置
